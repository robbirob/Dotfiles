# Sway Keybinds

Mod = Super/Windows key

## Launch / Session
- Mod+Return -> Terminal (kitty)
- Mod+d -> App launcher (bemenu)
- Mod+Shift+q -> Close window
- Mod+Shift+e -> Exit Sway
- Mod+Shift+c -> Reload config

## Layout / Modes
- Mod+f -> Fullscreen toggle
- Mod+b -> Split horizontal
- Mod+v -> Split vertical
- Mod+s -> Stacking layout
- Mod+w -> Tabbed layout
- Mod+e -> Toggle split layout
- Mod+Space -> Toggle focus mode (tiling/floating focus)
- Mod+Shift+Space -> Toggle floating

## Focus / Move
- Mod+h/j/k/l -> Focus left/down/up/right
- Mod+Shift+h/j/k/l -> Move left/down/up/right
- Mod+Tab -> Focus next
- Mod+Shift+Tab -> Focus prev

## Workspaces
- Mod+1..0 -> Switch to workspace 1..10
- Mod+Shift+1..0 -> Move container to workspace 1..10

## Audio
- Mute -> Toggle mute (XF86AudioMute)
- Volume up -> Raise volume (XF86AudioRaiseVolume)
- Volume down -> Lower volume (XF86AudioLowerVolume)

## Autostart
- Waybar is started via `exec waybar` in Sway's extraConfig.
